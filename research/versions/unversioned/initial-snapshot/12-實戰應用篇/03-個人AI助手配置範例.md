## 1. 配置哲學

```
OpenClaw 個人助手的配置哲學：

1. 「只有我一個人用」
   → 不需要多租戶
   → 不需要複雜的權限管理
   → 安全模型以自用為主

2. 「記住一切」
   → 長期記憶是核心價值
   → 記憶應該自然累積，不需要手動管理
   → 跨通道記憶共享

3. 「個性化到極致」
   → SOUL.md 反映我理想中的 AI 夥伴
   → 說話風格適合我的溝通習慣
   → 技能是我日常需求的延伸

4. 「可靠且低維護」
   → 啟動後就忘記它的存在
   → 自動處理更新和備份
   → 錯誤時優雅降級
```

```json5
      type: "openai",

      default_model: "gpt-4o",
      rate_limit: {
        max_requests_per_minute: 60,
        max_tokens_per_hour: 500000
      }
    },
    anthropic: {
      type: "anthropic",
      apiKey: "${ANTHROPIC_API_KEY}",
      models: ["claude-3-opus", "claude-3-sonnet"],
      default_model: "claude-3-sonnet"

    ollama: {
      type: "ollama",
      base_url: "http://localhost:11434",
      models: ["llama3", "codellama"],
      default_model: "llama3"
    }
  },

  // ===== Agents =====
  agents: {
    // 主助手
    claw: {
      model: "openai/gpt-4o",
      persona: "./personas/claw/SOUL.md",
      user_profile: "./personas/claw/USER.md",
      memory_guide: "./personas/claw/MEMORY.md",
      heartbeat: "./personas/claw/HEARTBEAT.md",
      temperature: 0.7,
      max_tokens: 4096,
      max_context_tokens: 128000,  // GPT-4o 的完整上下文
      allowed_tools: [
        "search_web", "memory_read", "memory_write",
        "weather_query", "calendar_read", "calendar_write",
        "remind_set", "translate", "note_create",
        "daily_briefing", "bookmark_save", "learning_log",
        "habit_track", "file_read"

      denied_tools: ["exec"],  // 不直接執行指令

      model: "anthropic/claude-3-sonnet",
      persona: "./personas/code-buddy/SOUL.md",
      temperature: 0.3,

      allowed_tools: [
        "file_read", "file_write", "exec",
        "search_web", "memory_read", "memory_write"
      ],
      sandbox: {
        enabled: true,
        backend: "docker",
        image: "openclaw/sandbox:node20"
      }
    },

    // 語音社交
    "voice-friend": {
      model: "openai/gpt-4o",
      persona: "./personas/voice-friend/SOUL.md",
      temperature: 0.85,
      max_tokens: 300,  // 語音要短
      allowed_tools: [
        "memory_read", "memory_write",
        "search_web", "weather_query", "translate"
      ]
    }
  },

  // ===== 通道 =====
  channels: {
    discord: {
      enabled: true,
      token: "${DISCORD_BOT_TOKEN}",
      default_agent: "claw",
      voice: {
        enabled: true,
        agent: "voice-friend",
        stt: { engine: "whisper", language: "zh-TW" },
        tts: {
          engine: "azure",
          voice: "zh-TW-HsiaoChenNeural",
          style: "chat"
        }
      },
      // 特定頻道對應特定 Agent
      channel_routing: {
        "coding-channel-id": "code-buddy",
        "general-channel-id": "claw"
      }
    },
    telegram: {
      enabled: true,
      token: "${TELEGRAM_BOT_TOKEN}",
      default_agent: "claw"
    },
    api: {
      enabled: true,
      auth: { type: "bearer", token: "${API_TOKEN}" }
    }
  },

  // ===== 記憶 =====
  memory: {
    backend: "sqlite",
    path: "./data/memories/claw.db",
    auto_memorize: true,
    per_user: false,  // 個人使用，不需要分使用者
    embedding: {
      provider: "openai",
      model: "text-embedding-3-small"
    },
    encryption: {
      enabled: true,
      algorithm: "aes-256-gcm"
    },
    backup: {
      enabled: true,
      schedule: "0 3 * * *",  // 每天凌晨 3 點
      path: "./data/backups/",
      keep: 30  // 保留 30 天
    }
  },

  // ===== 安全 =====
  security: {
    global: {
      allow_tools: true,
      require_approval_for: []  // 自用不需要人工批准
    }
  },

  // ===== 日誌 =====
  logging: {
    level: "info",
    file: "./logs/gateway.log",
    rotation: {
      max_size: "50M",
      max_files: 10
    }
  },

  // ===== Heartbeat（自主任務） =====
  heartbeat: {
    enabled: true,
    schedules: {
      "daily-briefing": {
        cron: "0 8 * * *",   // 每天早上 8 點
        agent: "claw",
        task: "Generate daily briefing",
        channel: "telegram"  // 發送到 Telegram
      },
      "memory-review": {
        cron: "0 23 * * 0",  // 每週日晚上 11 點
        agent: "claw",
        task: "Review and consolidate weekly memories"
      },
      "backup-check": {
        cron: "0 4 * * *",
        agent: "claw",
        task: "Verify memory backup integrity"
      }
    }
  }
}
```

## 4. SOUL.md 人格設計

```markdown
# Claw — Eason 的個人 AI 夥伴

## 核心身份
你是 Claw，Eason 的個人 AI 夥伴。你不是一個通用助手，而是專屬於 Eason 的
智慧夥伴。你了解他的工作、興趣、習慣和目標。你們之間是長期合作的關係。

## 性格
- **務實高效**：Eason 重視效率，你的回答應該直接切入重點
- **技術深度**：能深入討論技術話題，不需要過度解釋基礎概念
- **適度幽默**：偶爾的幽默可以，但不要影響效率
- **主動思考**：不只回答問題，還要提出你注意到的相關事項
- **坦率直接**：如果 Eason 的想法有問題，直接指出

## 說話風格
- 繁體中文為主，技術術語可用英文
- 簡潔為上，避免客套話（不需要「很高興幫助你」之類的）
- 使用代碼區塊展示技術內容
- 適當使用清單和結構化格式
- 可以用「你」稱呼 Eason

## 記憶使用
- 主動記住重要資訊，不需要被要求
- 在相關時自然地引用過去的對話
- 追蹤進行中的專案和任務
- 記住 Eason 的偏好和習慣

## 你知道的事
- Eason 是軟體工程師，專精 TypeScript 和 AI Agent 開發
- 目前在研究 OpenClaw 架構（你自己！）
- 使用 VS Code、Copilot CLI 作為主要開發工具
- 偏好 Linux 開發環境

## 特殊指令
- 當 Eason 說「記住」→ 一定要寫入長期記憶
- 當 Eason 說「提醒我」→ 設定提醒
- 當 Eason 說「摘要」→ 生成對話摘要
- 當 Eason 說「日報」→ 觸發每日摘要技能
```

## 6. MEMORY.md 記憶指引

```markdown
# 記憶管理指引

## 自動記憶規則
以下情況自動記住（不需要 Eason 明確要求）：

### 高優先級（必須記住）
- Eason 明確說「記住」的任何事情
- 新專案的名稱和目的
- 重要決策和原因
- 技術偏好的變化
- 截止日期和承諾

### 中優先級（應該記住）
- 正在研究的技術話題
- 遇到的問題和解決方案
- 表達的強烈喜好或厭惡
- 重複詢問的話題（表示重要）

### 低優先級（可以記住）
- 閒聊中提到的個人資訊
- 臨時的想法和靈感
- 推薦的文章或工具

## 記憶衛生
- 每週自動回顧記憶，合併重複項目
- 過時的資訊標記為 archived
- 矛盾的記憶保留最新的版本
- 不記住敏感資訊（密碼、token 等）

## 記憶格式
記憶內容應該簡潔、自足（不依賴上下文就能理解）：
- ✅ 「Eason 開始研究 OpenClaw 框架，目標是建構個人 AI 助手（2024-01）」
- ❌ 「他對那個很有興趣」（缺乏上下文）
```

## 8. 自訂技能

### 8.1 每日摘要技能

```typescript
// skills/daily-briefing/index.ts
export const dailyBriefingSkill = {
  name: "daily_briefing",
  description: "生成每日早晨摘要",
  parameters: {},
  handler: async () => {
    // 1. 行事曆
    const calendar = await googleCalendar.getToday();

    // 2. 天氣
    const weather = await weatherAPI.query("Taipei");

    // 3. 未完成任務（從記憶搜尋）
    const pendingTasks = await memoryStore.search({
      query: "pending task todo",
      category: "task",
      limit: 5
    });

    // 4. 組裝摘要
    return {
      calendar: calendar.events,
      weather: { temp: weather.temp, condition: weather.condition },
      pending_tasks: pendingTasks.map(t => t.content),
      date: new Date().toLocaleDateString("zh-TW")
    };
  }
};
```

### 8.2 書籤管理技能

```typescript
// skills/bookmark-manager/index.ts
export const bookmarkSkill = {
  name: "bookmark_save",
  description: "儲存書籤到個人收藏",
  parameters: {
    url: { type: "string", description: "網址" },
    title: { type: "string", description: "標題" },
    tags: { type: "array", items: { type: "string" }, description: "標籤" },
    note: { type: "string", description: "備註", optional: true }
  },
  handler: async (args) => {
    // 存到記憶系統
    await memoryStore.write({
      content: `書籤：[${args.title}](${args.url}) - ${args.note || ""}`,
      category: "bookmark",
      tags: args.tags,
      importance: 6
    });

    return { success: true, message: `已收藏：${args.title}` };
  }
};
```

### 8.3 學習追蹤技能

```typescript
// skills/learning-tracker/index.ts
export const learningLogSkill = {
  name: "learning_log",
  description: "記錄學習內容",
  parameters: {
    topic: { type: "string", description: "學習主題" },
    duration: { type: "number", description: "學習時間（分鐘）" },
    notes: { type: "string", description: "學習筆記" },
    resources: { type: "array", items: { type: "string" }, optional: true }
  },
  handler: async (args) => {
    const entry = {
      date: new Date().toISOString(),
      topic: args.topic,
      duration: args.duration,
      notes: args.notes,
      resources: args.resources || []
    };

    await memoryStore.write({
      content: `學習記錄 [${args.topic}] ${args.duration}分鐘：${args.notes}`,
      category: "learning",
      importance: 7,
      metadata: entry
    });

    // 統計本週學習
    const weeklyLearning = await memoryStore.search({
      category: "learning",
      since: getWeekStart(),
      limit: 100
    });
    const totalMinutes = weeklyLearning.reduce(
      (sum, m) => sum + (m.metadata?.duration || 0), 0
    );

    return {
      logged: entry,
      weekly_total: `${Math.floor(totalMinutes / 60)}h ${totalMinutes % 60}m`
    };
  }
};
```
